# xG-Wyscout
A basic expected goals model using Wyscout's publicly available event data:

Pappalardo, Luca; Massucco, Emanuele (2019): Soccer match event dataset. figshare. Collection. https://doi.org/10.6084/m9.figshare.c.4415000.v5 



data_functions.R: Contains multiple functions for processing the data and for plotting.

data_processing.R: A script to read in and process the data into usable format.

data_visualisation.R: A srcipt to plot the data pre and post model.

model.R: A script to fit a logistic regression model to the training data.
